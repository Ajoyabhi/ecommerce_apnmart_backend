## Packages
zustand | Lightweight global state management for Cart and Wishlist
embla-carousel-autoplay | Auto-playing hero banners for the homepage
lucide-react | Beautiful icons for the UI
framer-motion | Smooth animations for cart drawer and page transitions

## Notes
- State Management: Used Zustand for the Cart and Wishlist as it provides a simpler, more performant API for React 18 compared to Redux, while still meeting the requirement for a global store.
- Animations: Used Framer Motion for the Cart slide-out drawer and page transitions to ensure a premium feel.
- Carousel: Used Embla Carousel for the hero banner and product sliders.
- Images: Using high-quality Unsplash placeholders for the lifestyle and product imagery.
- Price formatting assumes the `price` field in the database is in cents/paise (integer), formatted to dollars.
